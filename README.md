# Wiki

## 目的

何かしらの設定手順や技術用語、知識を整理するための自分用の Wiki。</br>
調べたけどわけわからん事も少なくないので自分なりに噛み砕いた表現で記載する。


## ルール

- イメージしやすい説明を記載する。
- 用語説明の粒度が細かくなりそうなら別途ファイルを用意する。
