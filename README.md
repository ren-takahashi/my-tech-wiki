# My Tech Wiki

## 🎯 目的

知った用語や知識を整理するための自分用のWiki。
調べたけどわけわからん事も少なくないので自分なりに噛み砕いた表現で記述している。


## 🧱 ルール

- 初心者にもイメージしやすい説明を記述をする。
- 用語説明の粒度が細かくなりそうなら別途ファイルを用意する。

## 🗂 ディレクトリ構成（カテゴリ例）

- `setup/`：開発環境・ツール設定（VSCode、Dockerなど）
- `fundamentals/`：ネットワークやOSの基礎
- `programming/`：プログラミング言語ごと
- `security/`：認証・暗号・脆弱性など
- `glossary/`：用語の簡易説明集
- `notes/`：雑多なメモ、一時保存
