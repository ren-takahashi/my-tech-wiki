# VS Code セットアップ

目的: エディタ環境を素早く再現できるようにする。

## 前提
- OS: Linux/Mac/Windows のいずれか

## 手順
1. インストール
2. 日本語化 (Japanese Language Pack)
3. 推奨拡張機能の導入
4. 設定 (settings.json) の適用
5. キーバインド (keybindings.json) の適用

## 推奨拡張機能
- 例: ESLint / Prettier / GitLens / Docker / Remote Development

## 設定例 (抜粋)

```jsonc
  // ここに好みの設定を追記
}
```

## トラブルシュート
